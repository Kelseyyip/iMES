# Project Title

This repository contains the code for the paper:

**"Interactive multimodal disease risk evaluation using a self-service non-invasive device"**  
Ziqing Ye, Chenglong Zhang, Baoyuan Wu, and David Zhang,

Communications Medicine, 2026.

## Overview

This repository provides the core code used in the study. Some paths or parameters may need to be adapted before running the scripts.


## Requirements

The code was developed with:

- Python 3.10.9
- PyTorch v2.2.2
- NumPy v1.24.2
- scikit-learn v1.3.0
